Human-First, AI-Powered

We teach you to think like an engineer—and use AI like a power tool.

Think > Generate

You start with a plan: requirements, edge cases, and pseudocode. “Why” comments are graded. AI accelerates you—it doesn’t think for you.

Pro AI Workflows

Prompt patterns, test-driven chats, critique loops, and pair-programming with AI. Ship faster while keeping quality high and reasoning explicit.

Proof You Can Code

Weekly no-AI labs & closed-internet checkpoints (e.g., quicksort, BFS/DFS, REST CRUD). You’ll be able to deliver without AI—and excel with it.

AI Honor Code
  • Write a short plan before you prompt (goals, edge cases, tests).
  • Annotate any AI-assisted code with “why” comments.
  • Use tests to accept or reject AI output; don’t trust, verify.
  • Credit significant AI help in your README.
  • Clear authorship via Git history—we look at git blame.
